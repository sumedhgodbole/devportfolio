Projects

Online Prediction for Vision-based Active Pursuit using Domain Agnostic Offline Motion Model

Vision-based pursuit-evasion algorithms have been known to predict the future location of an evader in a given environment. In this paper, we propose the use of an encoder-decoder LSTM as a predictive model that can produce a more accurate estimation of the evader's future location when compared to conventional filtering-based methods. We evaluate the effectiveness of our approach by setting up agents in a MORSE environment. We also show that our method is adaptable across domains without the explicit need to retrain the prediction module by evaluating it in a CARLA environment.
(Submitted to IEEE ICRA'21)

Automatic Job Application Tracking using a Decision Tree Classifier

Hindered by COVID-19, we are all struggling to get hired. The number of applications one has to usually make just to get into an interview has nearly tripled owing to the undeniably trying circumstances. Machine Learning-based Email-Classification is used to separate the application receipt emails from regular emails and to collect them under one label. This facilitates automatic application tracking. The emails under a single label can potentially be exported as a single file and a fixed schedule automizing the task of job application tracking.

The Scope of Human Computer Interaction

Native Android application to help obtain information about any location towards which the device camera is pointed, resulting in an elevated usability index (HCI) for conventionally used Navigation Apps. It provides the user with a feature to pan the camera during which, an AR-based overlay containing information about the target location is displayed on the screen of the host device. Processing of the sensor data and the rendering the overlay was done by leveraging the Android Studio IDE. Our 4-member undergraduate team won first place in National Paper Presentation Threshold'17.
